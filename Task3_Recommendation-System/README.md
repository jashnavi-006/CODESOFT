# Movie Recommendation System

## Description
This project recommends movies based on genre similarity using a content-based filtering approach.

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Features
- Takes a movie name as input
- Recommends 5 similar movies
- Uses CountVectorizer and Cosine Similarity

## How to Run
1. Install dependencies:
   pip install pandas scikit-learn

2. Run the program:
   python recommendation.py
