# Rule-Based Chatbot

## Project Description
This is a simple rule-based chatbot developed using Python for the CODSOFT Artificial Intelligence Internship.

## Features
- Greets the user
- Responds to predefined questions
- Handles unknown inputs
- Ends the conversation when the user types "bye"

## Technologies Used
- Python
- Visual Studio Code

## How to Run
1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python chatbot.py
```

## Sample Inputs
- hello
- hi
- how are you
- your name
- what is ai
- thank you
- bye
