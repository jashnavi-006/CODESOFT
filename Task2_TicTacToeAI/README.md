# 🎮 Tic-Tac-Toe AI (CODSOFT Internship Task 2)

## 📌 Description
This project is a Tic-Tac-Toe game built using Python where a human player plays against an AI agent.

The AI uses the **Minimax algorithm** to make optimal moves and is unbeatable.

---

## 🚀 Features
- Human vs AI gameplay
- Intelligent AI using Minimax algorithm
- Win detection
- Draw detection
- Interactive console-based interface

---

## 🧠 AI Logic
The AI evaluates all possible future moves using the Minimax algorithm and selects the optimal move.

---

## 🛠 Technologies Used
- Python
- Minimax Algorithm
- Game Theory Basics

---
